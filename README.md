# Elm emoji picker

![Emoji picker, yeah](https://raw.githubusercontent.com/dawehner/elm-emoji/master/pictures/screenshot.jpg)


THis project is a emoji picker build in elm.

## Installation

* elm-package install
* npm install
* [Create Elm App](https://github.com/halfzebra/create-elm-app).

## Start it

```
elm-app start
```
